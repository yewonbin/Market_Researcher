# AI 보험·금융 트렌드 리서치 Agent — 구조 요약

> 보험·AI·핀테크 뉴스를 매일 자동으로 **수집 → 분류 → 요약 → 시사점 도출 → 발송**하는 에이전트

---

## 📌 프로젝트 요약 (포트폴리오용)

**AI Agent 기반 보험·금융 산업 트렌드 리서치 자동화**

- **AI Agent 기반 보험·금융 산업 트렌드 리서치 자동화** 파이프라인 설계·구현
- **n8n + LLM(Gemini) API** 를 활용해 보험산업 AI 트렌드 뉴스 **수집·요약·분류 자동화**
- LLM **Structured Output(JSON 스키마 + enum)** 으로 기사 분류 일관성·정확도 확보 (환각/오분류 방지)
- 수집 정보를 기반으로 **비즈니스 적용 가능 Use Case·시사점 자동 도출**
- **Google Sheets·이메일(HTML) 연동**으로 리서치 결과를 구조화하여 발송·축적 (*Notion 확장 가능*)
- **n8n / cron 스케줄러**로 매일 무인 자동 실행 구성
- **무료 LLM 등급 + 요청 속도 제어(RPM)·자동 재시도**로 운영비 0원 안정 운영

> **기술 스택**: Python · Google Gemini API(`google-genai`) · feedparser(RSS) · Jinja2 · Gmail SMTP · Google Sheets API · n8n

---

## ⭐ 주요 기능

| # | 기능 | 설명 |
|---|------|------|
| 1 | **뉴스 자동 수집** | RSS 피드(보험·인슈어테크·AI 키워드) 다중 소스 수집 |
| 2 | **중복 제거** | 동일 헤드라인(다른 매체)·기존 발송 기사 자동 필터링 |
| 3 | **AI 요약** | 기사별 핵심 내용을 한국어 2~3문장으로 요약 |
| 4 | **AI 분류** | 10종 카테고리로 자동 분류(Structured Output) |
| 5 | **시사점 도출** | 보험·금융 비즈니스 관점의 시사점 자동 생성 |
| 6 | **Use Case 제안** | 보험사가 적용 가능한 구체적 활용 방안 제시 |
| 7 | **관련도 스코어링** | 보험산업 관련도 1~5점 평가 후 임계값 미달 자동 제외 |
| 8 | **리포트 생성** | 카테고리별로 정리된 HTML 브리프 자동 작성 |
| 9 | **자동 발송·축적** | 이메일 발송 + Google Sheets 누적 저장 |
| 10 | **스케줄 자동화** | 매일 정해진 시각에 무인 실행 |

## 🔄 전체 동작 흐름

```
① RSS 수집      RSS 피드 7개에서 보험·AI·핀테크 뉴스 수집
② 신규 선별      이미 본 기사(seen.json)는 건너뛰기 → 중복 발송 방지
③ Gemini 분석    기사마다 분류+요약+시사점+UseCase+관련도 생성 (무료 AI)
④ 필터          관련도 기준 미달 기사 제외
⑤ HTML 리포트   카테고리별로 묶어 보기 좋은 HTML 생성
⑥ 발송          📧 이메일(Gmail) + 📄 Google Sheets 누적 저장
```

## 📁 파일 구조

| 파일 | 역할 |
|------|------|
| `run.py` | **메인 실행기** — ①~⑥ 전체를 순서대로 실행 (CLI) |
| `config.yaml` | 설정: 모델, RPM, RSS 피드, 카테고리, 기사 수 상한 |
| `.env` | 비밀값: **Gemini API 키**, Gmail 비밀번호, 시트 정보 |
| `src/collect.py` | RSS 수집 + 신규 선별 |
| `src/analyze.py` | **Gemini 분석** (속도제한·자동재시도 포함) |
| `src/report.py` | 카테고리별 HTML 리포트 빌드 |
| `src/state.py` | 중복 방지용 기록(`seen.json`) |
| `src/outputs/email_out.py` | Gmail SMTP 이메일 발송 |
| `src/outputs/sheets_out.py` | Google Sheets 누적 저장 |
| `n8n/workflow.json` | n8n 자동 스케줄 워크플로우 |

## ⚙️ 핵심 기술 포인트

- **무료 AI**: Gemini 무료 등급 → 운영비 0원
- **무료 한도 대응**: 분당 요청 한도(RPM)에 맞춰 요청 간격 자동 조절 + 429(한도 초과) 시 자동 재시도 → **기사 누락 없음**
- **구조화 출력**: AI 응답을 JSON 스키마로 고정 → 결과가 항상 일정한 형식
- **확장 용이**: `config.yaml`의 RSS 키워드·카테고리만 바꾸면 다른 주제로도 전환 가능

## ▶️ 실행 방법 (요약)

```powershell
# 1. 의존성 설치
pip install -r requirements.txt

# 2. 키 설정 (.env 의 GEMINI_API_KEY 에 무료 키 입력)
#    무료 키 발급: https://aistudio.google.com/apikey

# 3. 실행
python run.py --dry-run     # 발송 없이 HTML 미리보기
python run.py               # 실제: 수집→분석→이메일+시트 발송
```

자동 실행: Windows **작업 스케줄러** 또는 `n8n/workflow.json` 으로 매일 자동화.
